# Șantierul Cuvintelor

Un joc educațional de sinteză fonemică.

## Cerințe

- Python 3.x
- Pygame

## Instalare

1. Instalează dependențele:
   ```bash
   pip install -r requirements.txt
   ```

## Rulare

Pornește jocul executând:
```bash
python main.py
```

## Cum se joacă

1. Literele vor veni pe banda rulantă.
2. Trage de vagoane (Drag & Drop) pentru a le aranja în ordinea corectă și a forma cuvântul "CASA".
3. Apasă butonul "Asamblează" pentru a verifica.
